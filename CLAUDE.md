# scottastrophic.com

Personal website for Scott Koenig (scottastrophic).

## Stack
- Vite + React + TypeScript + Tailwind CSS
- Node 25.x, npm 11.x

## Dev
- `npm run dev` — starts dev server on http://localhost:5173/
- `npm run build` — production build to `dist/`

## Project context
- This site serves as a demonstration of what can be built with AI-assisted coding
- Design direction: clean, straightforward, and professional
- Social links: GitHub (scottastrophic), LinkedIn (scottastrophic)
- Domain: scottastrophic.com

## Conventions
- Use Tailwind utility classes for styling (no CSS modules)
- Custom keyframe animations live in `src/App.css`
- Keep it simple — this is a personal landing page, not an enterprise app
